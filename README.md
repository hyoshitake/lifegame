# lifegame
2009年ごろの学校課題で作成したライフゲームです。

## デモ
![デモ画像](https://raw.githubusercontent.com/hyoshitake/lifegame/master/demo.gif "デモ画像")


## 使用ライブラリ
- DXライブラリ
https://dxlib.xsrv.jp/

## 使い方
lifegame.exeを起動して下さい
- カラーパレットをクリックすると色を変更できます
- キャンバスにドラッグで線を描きます
- 「S」キーで初期化です
- 「A」キーで動作を停止し、入力モードになります
- 「G」キーでグリッド線を表示します
- 「エンター」キーでスタートです
